Introduccion a TuPay API

Bienvenido a la documentacion de la API de TuPay Finance. Esta guia te ayudara a integrar pagos entrantes (PayIn) y salientes (PayOut) en tu aplicacion de forma segura y eficiente.

Que es TuPay API?

TuPay API es una interfaz REST que te permite procesar transacciones financieras en Guatemala y Republica Dominicana. Con nuestra API puedes:

  • Recibir pagos (PayIn): Genera enlaces de pago seguros para tus clientes
  • Enviar fondos (PayOut): Realiza transferencias a cuentas bancarias o billeteras digitales
  • Consultar estados: Verifica el estado de cualquier transaccion en tiempo real
  • Gestionar webhooks: Recibe notificaciones automaticas de cambios de estado

Caracteristicas Principales

Seguridad

Autenticacion mediante API Key con encriptacion SSL/TLS en todas las comunicaciones.

Velocidad

Procesamiento en tiempo real con respuestas en milisegundos.

Confiabilidad

99.9% de uptime con infraestructura redundante y monitoreo 24/7.

Multi-tenant

Soporte nativo para multiples paises y monedas con aislamiento de datos.

Entornos Disponibles

URLs por País

TuPay API opera en diferentes URLs según el país:

  • Guatemala: https://api-guatemala.tupay.finance
  • República Dominicana: https://api-rd.tupay.finance
